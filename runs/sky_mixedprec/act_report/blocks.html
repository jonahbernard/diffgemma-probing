block / fp4 cost

magenta/red = mxfp4 (W4A4), green = bf16. Outliers inflate fp4 block scales, so big per-block absmax = more quantization error.
What am I looking at?
fp4 splits each activation row into contiguous blocks of 32 channels and gives each block ONE shared scale, set by that block's largest magnitude (its absmax). One panel per GEMM site. Each line tracks, at each denoising step (x-axis), the worst block's absmax averaged over all 30 layers (log y-axis). Higher = a bigger scale forced on a block, so its 31 other (smaller) values get quantized more coarsely -- i.e. more fp4 error. green = bf16, red = mxfp4. Watch how the red line moves across denoising steps and how the sites differ: the site/step where red is highest is where fp4 hurts most.

Higher per-block absmax = larger fp4 scale for that block = coarser quantization of its other (small) values. Averaged over layers at each denoising step.
