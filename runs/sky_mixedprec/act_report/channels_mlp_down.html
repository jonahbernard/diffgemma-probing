magenta/red = mxfp4 (W4A4), green = bf16. Outliers inflate fp4 block scales, so big per-block absmax = more quantization error.
What am I looking at?
Each curve is the per-channel absmax of the activation feeding the mlp.down GEMM at ONE layer/step: for every one of the H hidden channels (x-axis), the largest magnitude that channel reaches across all tokens. Left plot = value at each channel index, so a tall spike is an 'outlier channel' (a specific hidden dim that is persistently large). Right plot = the exact same numbers sorted high-to-low on a log y-axis, which shows how few channels carry the extremes and how far above the rest they sit. green = bf16, red = mxfp4; where red towers over green, quantization is facing a bigger outlier. A few representative layers are shown (one row each).

Left: per-channel absmax across the hidden dim (spikes = outlier channels). Right: same values sorted descending on a log scale (how few channels dominate, and by how much).
