heatmap: median per-block absmax

magenta/red = mxfp4 (W4A4), green = bf16. Outliers inflate fp4 block scales, so big per-block absmax = more quantization error.
What am I looking at?
The MEDIAN block-absmax across all blocks in the tensor -- what a typical block's fp4 scale is set by. Contrast with 'max per-block absmax': the gap between them is how much the worst block is inflated vs normal.

x = denoising step, y = decoder layer. diff = mxfp4 - bf16 (red = mxfp4 larger).
